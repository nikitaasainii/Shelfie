# 📚 Bookshelf

A personal book tracking web app with a cozy, visual bookshelf interface.

## What it does

- Track books you're reading, have finished, or want to read
- Visual bookshelf — each book appears as a spine on a shelf
- Click a spine to see the cover, click the cover to see details
- Search and add books via Google Books API
- Rate books, set status, and add personal notes

## Tech Stack

| Layer | Tech |
|---|---|
| Frontend | React + Vite |
| Styling | Tailwind CSS |
| Backend | Python + FastAPI |
| Database | PostgreSQL |
| Book Data | Google Books API |

## Project Structure

```
bookshelf/
├── frontend/      # React app
└── backend/       # FastAPI app (coming soon)
```

## Getting Started

```bash
cd frontend
npm install
npm run dev
```

App runs at `http://localhost:5173`
